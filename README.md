# Deadlock Detection in Operating Systems

## 📌 Description
This project simulates a multiprogramming environment and detects deadlocks using:

- Resource Allocation Graph (RAG)
- Wait-For Graph (WFG)

## ⚙️ Features
- Detects deadlock using two methods
- Identifies deadlocked processes
- Uses DFS for cycle detection

## 🛠️ How to Run
```bash
gcc deadlock.c -o deadlock
./deadlock
